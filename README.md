# 🚀 .NET CRUD App (Full Stack)

A full-stack CRUD application built using .NET Web API and React.

## ✨ Features
- Create, Read, Update, Delete operations
- RESTful API (.NET)
- React frontend UI
- API integration with frontend

## 🛠 Tech Stack
- Backend: .NET Web API
- Frontend: React.js
- Database: (add yours)

## 📂 Project Structure
- /CURDDemo → Backend API
- /Frontend → React App

## ⚡ Run Locally

### Backend
cd CURDDemo  
dotnet run  

### Frontend
cd Frontend  
npm install  
npm start
